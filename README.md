# Full-Stack Login Application

## 📌 Objective
This project is a simple Full-Stack Login Application built using React (Frontend) and Node.js with Express (Backend).

---

## 🛠 Tech Stack
- Frontend: React (Functional Components + Hooks)
- Backend: Node.js + Express
- Routing: React Router v6
- API Calls: Axios
- Styling: CSS

---

## 🚀 Features
- Login page with username and password
- Backend API validation (POST /login)
- Correct credentials: admin / admin
- Error message for invalid login
- Username remembered using localStorage
- Protected Welcome page
- Logout functionality
- Proper HTTP status codes (200, 401)

---

## ▶ How to Run

### Backend
cd backend  
npm install  
node server.js  

### Frontend
cd frontend  
npm install  
npm start
